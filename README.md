# LAN Communication System

A multi-user LAN communication application supporting **video calls, audio calls, screen sharing, group chat, and file transfer** using a **client-server architecture**.

Built with **Python sockets, OpenCV, PyAudio, and Tkinter**.

---

## Installation

Install dependencies:

```
pip install -r requirements.txt
```

Or:

```
pip install opencv-python numpy Pillow pyaudio
```

---

## Usage

Start the server:

```
python server.py
```

Start the client:

```
python client.py
```

Enter the **server IP address** and **username** to connect.

---

## Troubleshooting

**Connection issues**

* Ensure server is running
* Verify server IP
* Confirm devices are on the same network

**Video issues**

* Check webcam connection
* Verify OpenCV installation

**Audio issues**

* Check microphone permissions
* Verify PyAudio installation
